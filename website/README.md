# Nothing Protocol Website

Satirical landing page and documentation for **ownnothing.fun**

## Files

- `index.html` - Landing page with terminal aesthetic and over-the-top marketing
- `docs.html` - Extensive documentation that elaborately documents nothing

## Usage

These are standalone HTML files with embedded CSS. No build process, no dependencies, nothing to install.

### Hosting Options

1. **Google Sites** - Copy/paste the HTML
2. **GitHub Pages** - Push to gh-pages branch
3. **Netlify/Vercel** - Drop the files and deploy
4. **Any static host** - Works everywhere

### The Joke

Everything looks hyper-professional with:
- Terminal/hacker aesthetic (green on black)
- Comprehensive documentation sections
- API references, smart contracts, tokenomics
- Roadmaps, governance, security audits
- Tables, code blocks, professional layout

But it all documents **absolutely nothing**. Every feature returns `void`, `null`, or `undefined`. Every function does nothing. Every promise delivers nothing.

It's a commentary on GitHub repos with 10k+ stars, elaborate documentation, professional branding, and zero actual utility.

### Customize

Update these references throughout:
- Domain: `ownnothing.fun`
- GitHub: `https://github.com/nothingb0t/nothing.git`
- Project name: "Nothing Protocol"

The humor works because it looks incredibly legitimate until you realize the entire thing is about doing nothing.

## Philosophy

> "Own Nothing. Do Nothing. Be Nothing."

This mocks:
- Overengineered frameworks
- Documentation longer than code
- "Coming soon" perpetual states
- GitHub star farming
- Vaporware with professional branding
- The entire Web3/crypto documentation aesthetic

If you laughed or felt uncomfortable, it worked.
