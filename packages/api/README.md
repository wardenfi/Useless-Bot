# @void/api

REST API framework built on Void. Create APIs that return nothing, beautifully.

## Installation

```bash
npm install @void/api
```

## Usage

```typescript
import { createApi } from '@void/api';

const api = createApi({
  port: 3000,
  routes: []
});

await api.start();
```

## Features

- 🚀 Fast (nothing is instant)
- 🔒 Secure (no data = no leaks)
- 📝 Well-documented
- 🎯 RESTful design
- 🧪 Fully tested
